# NexWork Job Portal Website

## Overview

JobSeeking is a modern Job Portal Website built using **HTML and CSS only**.
The purpose of this project is to create a clean, user-friendly platform where users can explore job opportunities, view companies, and understand how the hiring process works.

This project follows the requirement of using **pure HTML and CSS without any libraries or frameworks**.

## Goals of the Website

* Provide a simple and modern job portal interface
* Help users explore job opportunities easily
* Show companies and available positions
* Create a professional multi-page layout using only HTML and CSS
* Improve front-end design and layout skills
